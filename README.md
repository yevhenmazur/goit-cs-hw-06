# goit-cs-hw-06
Socket Programming та комунікації реального часу

## Build manualy
```
 docker build -t goit-cs-hw-06 ./
```

## Run 
```
docker run -d -p 3000:3000 --rm goit-cs-hw-06
```

## Run in compose
```
docker compose up
```
